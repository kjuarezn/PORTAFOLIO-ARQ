Oficina de Arquitectura

El proyecto se encuentra ubicado en el municipio de San Lucas en el departamento de Sacatepéquez. El terreno contiene unas medidas de 4 mts de frente por 10 mts de fondo. La oficina cuenta con un área de recepción y de desinfección por el COVID-19. Baños generales, una oficina para los dibujantes, una oficina para el calculista, la oficina principal del arquitecto y cuenta con un show room para poder mostrar los proyectos a los clientes.

Sistema Constructivo

Muro de block

El tipo de sistema constructivo tradicional se encuentra conformado por una serie de zapatas, un cimiento corrido que amarrará toda la estructura seguidamente unas hiladas de block una solera de húmedad, solera intermedia, una solera de corona, columnas de hierro con concreto, y el block que se encuentra entre las soleras y columnas.

Planta Arquitectónica
Nivel 1

En el primer nivel encontramos primeramente el ingreso a través de una cámara de desinfección para poder eliminar cualquier tipo de virus antes del ingreso hacía la recepción. La recepción tiene la capacidad para dos personas y una secretaria. Seguidamente en la parte central encontramos un jardín para aprovechar la iluminación natural. En el siguiente modulo encontramos un modulo de servicio sanitario y una pequeña oficina para dos dibujantes y un espacio para la colocación de un plotter.

Planta Arquitectónica
Nivel 2

En el segundo nivel encontramos primeramente la oficina del arquitecto con vista hacia la calle, tiene una capacidad para dos personas manteniendo el espacio adecuado por el virus.En el modulo derecho nos encontramos con un modulo de servicio sanitario y una oficina para el calculista teniendo como iluminación natural la captación de la parte central del proyecto..

Planta Arquitectónica
Nivel 3

En el tercer nivel encontramos primeramente un área de descanso con vista hacía el exterior, en esta parte los empleados podrán ingerir sus alimentos manteniendo un distanciamiento adecuado y una circulación de aire natural. El complejo tiene un área específica para demostración de proyectos llamado show room la función principal de esta oficina es la de poder mostrarle al cliente como quedara el proyecto y de esta manera poder cerrar el trato de una manera más rápida.

Sección Longitudinal

La sección logitudinal que abarca todo el largo del proyecto pasando por recepción el módulo de gradas y las oficinas. Se puede observar el tipo de construcción la cuál es tradicional de mampostería. El modulo de gradas van empotradas en la pared para poder darle una mayor vista al espacio y lograr que de esta manera una especia de celosia. Cada oficina cuenta con un seperador llamado plexiglass para poder evitar un contacto directo con las personas y al mismo tiempo poderlas observar. en la parte del área de descanso se encuentra una pergola con una enredadera para proporcionar un área de relajamiento para el personal.

Vista Interior
Recepción

La recepción cuenta con una cámara de desinfección para poder contener la propagación de la pandemia. Cuenta con una plaza de tres butacas para el área de espera y una recepcionista. Se puede observar que en todo el área se encuentra conectada con el exterior a través de una serie de ventanerias. La cuál unicamente no ayuda para una correcta iluminación sino que de igual manera ayuda a que las personas puedan observar hacía su interior para poder atraer a clientes potenciales.

Vista Interior
Oficina del Arquitecto

La oficina del arquitecto consta de un escritorio princpial protegido por un vidrio de policarbonato transparente para protegerlo de las gotas directas que puedan contener el virus covid-19. La oficina tiene la capacidad para que el arquitecto pueda atender a dos personas manteniendo el distanciamiento adecuado. En la parte exterior cuenta con vista hacía la calle con un tipo de celosia para la protección de la luz solar. El tipo de piso es cerámico y en una de las partes del muro se implementó machimbre para hacer una combinación de texturas.
